# SpringBootLearning
## 简介
这是作者的一个SpringBoot学习项目，<br/>
* 集成了Redis作为缓存中间件，<br/>
* 集成MySQL作为数据存储层，<br/>
* 旨在对SpringBoot的学习和其他组件的集成

## 需要
* 基于 `JDK1.8`, `SpringBoot 2.5.6`
* `Maven3.3.9`构建(`可选`). 
* `Redis`
* `MySQL`

## 阅读    
读者可以在线观看也可以使用`git clone`拉取到本地阅读，本项目的所有程序都可以在`JDK1.8`，`Maven3.3.9`环境下编译通过，正常运行
